Tu travailles dans mon projet de site statique “HydroFacile”, dédié à l’hydroponie pour débutants en appartement.

Objectif :
Créer et intégrer un nouvel article SEO complet, utile et cohérent avec le site, en choisissant toi-même le meilleur sujet après analyse du projet existant.

Tu dois agir comme un éditeur SEO senior + intégrateur de site statique :
- analyser les contenus déjà présents ;
- identifier une opportunité éditoriale pertinente ;
- créer un article naturel, utile et bien structuré ;
- l’intégrer proprement dans le projet ;
- mettre à jour uniquement les fichiers nécessaires ;
- vérifier que rien d’important n’est cassé.

────────────────────────
1. ANALYSE DU SITE
────────────────────────

Commence par analyser le projet existant :
- articles déjà publiés ;
- titres H1/H2/H3 ;
- sujets et mots-clés déjà couverts ;
- structure des fichiers d’articles ;
- métadonnées SEO ;
- maillage interne ;
- images utilisées ;
- sitemap, flux RSS, index d’articles, catégories/tags ou fichiers de données s’ils existent ;
- ton éditorial global ;
- conventions de structure et d’intégration du projet.

Identifie :
- les sujets déjà bien couverts ;
- les angles proches mais incomplets ;
- les risques de doublon ou de cannibalisation ;
- les opportunités de longue traîne SEO ;
- les sujets adaptés à des débutants en hydroponie en intérieur ;
- les sujets concrets, utiles et faciles à relier aux articles existants.

Choisis ensuite le meilleur sujet à créer maintenant.

Critères de choix :
- sujet précis, pas trop large ;
- utile pour un lecteur français débutant ;
- adapté à l’hydroponie en appartement ;
- potentiel SEO réaliste ;
- complémentaire aux articles existants ;
- facilement maillable ;
- pas de doublon évident avec un article déjà publié.

Ne me demande pas de choisir le sujet. Décide en autonomie.

────────────────────────
2. NOTE DE DÉCISION
────────────────────────

Avant d’écrire l’article, ajoute dans ton récap final une courte note indiquant :
- sujet retenu ;
- mot-clé principal ;
- mots-clés secondaires ;
- intention de recherche ;
- raison du choix ;
- articles existants pertinents pour le maillage interne ;
- éventuels sujets écartés et pourquoi, si utile.

────────────────────────
3. EXIGENCE FORTE SUR LE TITRE
────────────────────────

Le titre doit être SEO-friendly, mais surtout naturel et agréable à lire.

Règles :
- éviter les titres qui ressemblent à une suite de mots-clés ;
- éviter les formulations lourdes, mécaniques ou sur-optimisées ;
- privilégier un français fluide, clair et humain ;
- choisir un titre qui fonctionne à la fois sur Google, dans une carte d’article et dans la page ;
- éviter les titres inutilement longs ;
- garder une vraie promesse de lecture.

Avant de retenir le titre final, vérifie :
- est-ce qu’il sonne bien à voix haute ?
- est-ce qu’il donne envie d’être lu ?
- est-ce qu’il semble écrit pour un humain ?
- est-ce qu’il reste bon pour le SEO sans être artificiel ?

Tu peux réfléchir à plusieurs variantes en interne, mais ne garde que la meilleure.

────────────────────────
4. RÉDACTION DE L’ARTICLE
────────────────────────

Rédige un article complet, original, naturel et utile.

Style attendu :
- français naturel ;
- ton simple, clair, pédagogue et rassurant ;
- pas de phrases creuses ;
- pas de remplissage SEO ;
- pas de sur-optimisation ;
- conseils réellement exploitables ;
- vocabulaire accessible aux débutants ;
- cohérence avec le branding HydroFacile.

Public visé :
Débutants en hydroponie en appartement, sans expertise technique.

Structure attendue :
- H1 naturel et optimisé ;
- introduction engageante ;
- sections H2/H3 bien organisées ;
- paragraphes lisibles ;
- listes à puces uniquement si elles améliorent la clarté ;
- conseils pratiques ;
- erreurs à éviter si pertinent ;
- FAQ courte uniquement si elle apporte une vraie valeur SEO ;
- conclusion naturelle avec ouverture vers d’autres contenus du site.

La longueur doit être adaptée au sujet et au niveau des autres articles du site.
Ne gonfle pas artificiellement le texte.

N’invente pas de faits précis non vérifiables.
Évite les affirmations trop catégoriques sur les rendements, la croissance des plantes, les économies d’eau ou les résultats obtenus.

────────────────────────
5. SEO ON-PAGE
────────────────────────

Optimise l’article avec :
- title SEO unique ;
- meta description naturelle et engageante ;
- slug propre ;
- structure Hn cohérente ;
- intégration naturelle du mot-clé principal ;
- variantes sémantiques pertinentes ;
- alt text d’image ;
- canonical si la structure du site l’utilise ;
- Open Graph / Twitter cards si le projet les utilise ;
- schema.org Article ou BlogPosting si le site l’utilise déjà.

Priorité :
- intention de recherche ;
- lisibilité ;
- utilité réelle ;
- cohérence éditoriale ;
- potentiel de clic depuis Google.

Le title SEO et le H1 peuvent être proches, mais ils ne doivent pas devenir lourds ou artificiels.

────────────────────────
6. MAILLAGE INTERNE
────────────────────────

Ajoute des liens internes naturels :
- depuis le nouvel article vers les articles existants les plus pertinents ;
- depuis 2 à 5 anciens articles vers le nouvel article si cela s’intègre naturellement.

Ne crée pas de liens forcés.
Ne modifie pas massivement d’anciens articles si ce n’est pas nécessaire.
Préserve le style des articles existants.

────────────────────────
7. IMAGE PRINCIPALE
────────────────────────

Prépare une image principale cohérente avec le sujet.

Règles :
- utiliser uniquement une image libre de droit avec source et licence vérifiables ;
- ne pas inventer de source, de crédit ou de licence ;
- ne pas réutiliser une image déjà présente dans le projet sauf nécessité claire ;
- privilégier une vraie photo crédible et naturelle ;
- éviter les illustrations génériques, visuels kitsch, images trop artificielles ou hors sujet ;
- choisir une image cohérente avec l’hydroponie en appartement et le sujet de l’article.

Si l’environnement permet de télécharger et intégrer proprement l’image :
- télécharge l’image ;
- optimise-la ou redimensionne-la si le workflow du projet le permet ;
- place-la dans le bon dossier ;
- utilise-la pour la vignette et l’image de l’article ;
- renseigne le texte alt ;
- ajoute le crédit/la source si le site le prévoit.

Si l’intégration complète n’est pas possible, prépare tout proprement avec :
- source recommandée ;
- URL de l’image ;
- licence ;
- crédit ;
- nom de fichier SEO conseillé ;
- texte alt ;
- emplacement prévu dans le projet.

────────────────────────
8. INTÉGRATION DANS LE SITE
────────────────────────

Crée réellement l’article dans le bon format utilisé par le projet.
Respecte exactement la structure existante.

Mets à jour uniquement les fichiers nécessaires, par exemple :
- listing des articles ;
- page d’accueil si elle affiche les derniers articles ;
- archive/blog si besoin ;
- sitemap ;
- flux RSS/Atom si présent ;
- fichiers JSON/YAML/JS/TS si utilisés ;
- navigation liée au blog ;
- catégories/tags si le site fonctionne ainsi ;
- image preview sociale si prévue ;
- liens internes depuis anciens articles.

Ne réorganise pas le projet.
Ne change pas la charte graphique.
Ne refactorise pas inutilement.
Fais des modifications propres, minimales et cohérentes.

────────────────────────
9. CONTRÔLES QUALITÉ
────────────────────────

Avant de finaliser, vérifie :
- absence de doublon évident avec un article existant ;
- absence de cannibalisation SEO évidente ;
- cohérence du ton avec HydroFacile ;
- qualité réelle du texte ;
- orthographe et grammaire ;
- lisibilité sur mobile si possible ;
- liens internes valides ;
- métadonnées complètes ;
- titre final naturel et agréable à lire ;
- image correctement intégrée ou documentée ;
- sitemap/RSS/index à jour si nécessaire ;
- absence de modification inutile.

Si le projet contient des scripts de build, validation, lint ou preview, lance-les si possible.
Si une commande échoue, corrige si c’est raisonnable, sinon explique clairement le problème.

────────────────────────
10. RÉCAP FINAL
────────────────────────

À la fin, donne un récapitulatif clair avec :
- sujet choisi ;
- mot-clé principal ;
- mots-clés secondaires ;
- intention de recherche ;
- raison du choix ;
- titre final retenu ;
- fichiers créés ;
- fichiers modifiés ;
- éléments SEO ajoutés ;
- liens internes ajoutés ;
- source, crédit et licence de l’image ;
- commandes de vérification lancées ;
- éventuelles limites ou actions manuelles restantes.

Règles importantes :
- décide en autonomie ;
- ne crée pas un sujet déjà traité sauf angle nettement différent ;
- ne rédige pas de contenu générique SEO ;
- ne casse pas la structure existante ;
- respecte le style existant ;
- privilégie la qualité réelle à la quantité ;
- si tu hésites entre plusieurs sujets, choisis le meilleur compromis entre utilité lecteur, cohérence éditoriale et potentiel SEO ;
- si tu hésites entre plusieurs titres, choisis celui qui paraît le plus naturel et agréable à lire.
