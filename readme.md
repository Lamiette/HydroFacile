# HydroFacile

URL cible : https://hydrofacile.fr

HydroFacile est un site statique sur l'hydroponie pour debutants en appartement.

Le projet garde une structure simple, facile a maintenir, avec un design propre et une navigation claire.
L'objectif est de publier ensuite des guides originaux pour expliquer la culture sans terre, le choix d'un petit systeme et les plantes les plus faciles pour commencer chez soi.

Points utiles :
- la structure du site reste volontairement simple et statique
- la regeneration des pages principales passe par `scripts/rebuild_articles.ps1`
- la ligne editoriale vise des contenus accessibles, concrets et rassurants
